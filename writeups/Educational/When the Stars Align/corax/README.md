# When the Stars Align...

We are presented with at beautiful image of a starry night.

![stars](../stars.png)

A clue to where the flag might be hidden is the pixels themselves. We know that the format is `EPT{....}`, and if we take a closer look at the pixels we see that the first three pixels are very close to each other in the Y-coordinate space. The fourth one however is further down. This is also the case in ASCII, the decimal values representing EPT are `69, 80, 84` and for the curly bracket we get the value `123`.

Below is a python script that first moves down the y-axis before moving one to the right on the x-axis. If we find a white pixel then we print out the y-value as character. 

```py
#! /usr/bin/python3

from PIL import Image

im = Image.open('stars.png')
pix = im.load()

for x in range(0, im.size[0]):
    for y in range(0, im.size[1]):
        if(pix[x, y][0] == 255):
            print(chr(y), end="")
```

If we run the script we get the flag.
